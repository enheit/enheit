## Neovim Plugins

A collection of "My Lovely" (ml*) plugins for Neovim, each designed to improve your editing experience:

- **[mltv](https://github.com/enheit/mltv)** - My Lovely Tree Viewer: A simple and elegant file tree viewer with intuitive vim-like navigation
- **[mltb](https://github.com/enheit/mltb)** - My Lovely Theme Builder: Generate beautiful, randomized themes using Tailwind CSS colors and color theory
- **[mlts](https://github.com/enheit/mlts)** - My Lovely Theme Selector: Interactive theme browser with live preview and persistent selection
- **[mlwf](https://github.com/enheit/mlwf)** - My Lovely Word Finder: Fast, live text search across your entire project with syntax highlighting
- **[mlfs](https://github.com/enheit/mlfs)** - My Lovely File Selector: Fuzzy file finder with smart filtering and beautiful highlighting
- **[mlau](https://github.com/enheit/mlau)** - My Lovely Autocomplete: Lightweight native completion plugin with LSP integration and function signatures

## Animation Libraries

JavaScript/TypeScript libraries for creating smooth, programmatic animations:

- **[animation-player](https://github.com/enheit/animation-player)** - A versatile animation player for JavaScript with support for SVG animations
- **[animation-player-tween](https://github.com/enheit/animation-player-tween)** - Tween helper library for smooth property interpolation and easing functions
